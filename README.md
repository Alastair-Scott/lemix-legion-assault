# WoW Legion Remix - Legion Assault Timer

A real-time timer application for tracking Legion Assault events in World of Warcraft: Legion Remix. Displays assault status, countdown timers, and start/end times for both EU and US servers.

## Assault Schedule

- **Assault Duration**: 6 hours
- **Wait Period**: 8.5 hours between assaults
- **Cycle Length**: 14.5 hours total

## Region Support

### EU Servers
- **Timezone**: UTC+1 (Server Time)
- **Status**: ✅ Fully tracked with reference time

### US Servers
- **Timezone**: UTC-5 (EST) - configurable
- **Status**: ⏳ Reference time pending
